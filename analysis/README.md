# Supply Chain Correlation Analysis

Email: 23f1000780@ds.study.iitm.ac.in

This repository contains the correlation analysis performed on a supply chain dataset with five key procurement and supplier performance metrics:

- Supplier_Lead_Time
- Inventory_Levels
- Order_Frequency
- Delivery_Performance
- Cost_Per_Unit

## Files Included

- **correlation.csv**  
  The correlation matrix generated using the CORREL() function in Google Sheets.

- **heatmap.png**  
  A heatmap visualization created using a Green–White–Red color scale (Green = high correlation, Red = low correlation) following Excel-style formatting guidelines.

## Purpose

This repository is prepared for validation as part of the Supply Chain Analytics: Correlation Matrix Visualization exercise.  
It demonstrates:

- Generation of a correlation matrix  
- Application of color-scale visualization  
- Proper export and documentation of results
